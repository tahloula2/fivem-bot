## All commands

* !id HEX --> Hexi writes the information of the person entered.
* !phone phoneNumber --> Writes the information of the person whose phone number is entered. A good way to find phone spammers.
* !ck HEX --> Hexi assigns CK to the person entered. Only **administrators** can use it.
* !wlekle HEX @User --> Hexi whitelists the entered person. (esx_whitelist)
* !wlsil HEX --> Hexi gets the whitelist from the person entered. (esx_whitelist)
* !paraver HEX AMOUNT --> Hexi sets the money of the person entered to the specified amount, **DOES NOT ADD!**
* !bankaparaver HEX AMOUNT --> Hexi adjusts the money of the person entered in the bank to the specified amount, **DOES NOT ADD!**
* !add money HEX AMOUNT --> the amount entered in the current money of the person whose hexi is entered **ADDITIONS**
* !add bank money HEX AMOUNT --> Amount entered in the current money of the person whose hexi is entered in the bank **ADDITIONS**
* !professional information profession**CODE** LEVEL (if not entered **0**) --> Shows the information of the profession according to the entered information (Code, salary, etc.)
* !professional HEX occupation**CODE** LEVEL --> Hexi changes the occupation of the person entered.
* !telnochange HEX NUMBER --> Hexi changes the phone number of the person entered. Only **administrators** can use it.
* !change name HEX NAME SURNAME --> Hexi changes the name and surname of the person entered.
* !invoiceinfo HEX --> Hexi shows all invoices of the person entered.
* !change license plate to be_license_plate --> Changes the entered license plate.
* !inventoryinfo HEX --> Hexi displays all items in the person's inventory.
